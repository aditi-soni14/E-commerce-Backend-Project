# E-commerce Backend Project

## Project Overview
**E-commerce Backend Project** is a RESTful API built with **Node.js**, **Express**, and **MongoDB**.  
It serves as the backend for an e-commerce platform, handling product management, user registration/login, and is designed for easy integration with any frontend application.

**GitHub Repository:** [E-commerce Backend Project](https://github.com/aditi-soni14/E-commerce-Backend-Project)

---

## Features
- CRUD operations for products (Create, Read, Update, Delete)
- User registration and login management
- Modular and scalable folder structure
- Ready for integration with frontend

---

## Tech Stack
- Node.js, Express.js
- MongoDB (with Mongoose)
- dotenv for environment variables

---

## Getting Started
Follow these steps to run the project locally:

```bash
# 1. Clone the repository
git clone https://github.com/aditi-soni14/E-commerce-Backend-Project.git
cd E-commerce-Backend-Project

# 2. Install dependencies
npm install

# 3. Create a .env file in the root directory with the following:
#    (replace with your own MongoDB connection string)
MONGO_URI=<Your MongoDB Connection String>
PORT=8080

# 4. Start the server
node index.js

# Your API will now run on http://localhost:8080
```
